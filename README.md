# Dashboard para um empresa de petshop
O repositório foi criado para apresentar um Dashboard criado para um cliente que é dona de um petshop. 

## Métricas visualizadas:
- Faturamento (total).
- Média dos pets.
- Gêneros que mais compra.
- Faturamento (tempo).
